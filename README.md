# Trilo - 极简划词翻译工具

Trilo 是一款专注于**速度**与**极简体验**的跨平台（Windows / macOS / Linux）桌面划词翻译工具。它安静地运行在后台，没有复杂的界面干扰，只有在你需要的时候，它才会瞬间出现。

![Trilo Preview](static/favicon.png)

## ✨ 主要特性

*   **极简设计**：无边框悬浮窗，专注于翻译内容本身。
*   **即时响应**：选中即翻，毫秒级响应速度。
*   **干扰极低**：翻译窗口自动适应内容大小，失去焦点自动消失。
*   **高度定制**：支持自定义快捷键、深色/浅色主题切换。

## 🚀 如何使用

1.  **选中文本**：在任意软件（浏览器、文档、编辑器等）中，用鼠标选中你想要翻译的文本。
2.  **按下快捷键**：按下默认快捷键 `Ctrl + Q`（可在设置中修改）。
3.  **查看结果**：翻译结果会立即显示在鼠标附近。
4.  **复制/关闭**： 
    *   **复制**：直接选中文本按 `Ctrl + C` 即可复制翻译结果。
    *   **关闭**：点击窗口以外的任意区域，或者再次按下快捷键，窗口即会自动消失。

## ⚙️ 配置指南 (必读)

Trilo 使用**小牛翻译 (Niutrans)** 的翻译引擎。为了正常使用，你需要配置自己的 API Key。不用担心，这是免费的。

### 第一步：获取 API Key
1.  访问 [小牛翻译开放平台](https://niutrans.com/) 并注册账号。
2.  登录控制台，创建一个新应用（或使用默认应用）。
3.  在应用管理页面，找到你的**文本** **`APP ID`** 和 **`API Key`**。


### 第二步：在 Trilo 中配置
1.  在电脑右下角的系统托盘中找到 Trilo 的图标（一个小圆圈）。
2.  **右键点击**图标，选择 **“设置”**。
3.  在设置面板中：
    *   将刚才获取的 `APP ID` 填入对应输入框。
    *   将 `API Key` 填入对应输入框。
4.  点击 **“保存”** 按钮。

现在，你就可以开始享受丝滑的翻译体验了！

## 🎨 其他设置

*   **自定义快捷键**：在设置面板点击快捷键按钮，直接按下你习惯的组合键（如 `Alt + Space`），软件会自动识别。
*   **主题切换**：支持浅色模式、深色模式，或跟随系统自动切换。
*   **开机自启**：开启后，Trilo 会随系统启动，随时待命。
