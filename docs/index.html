🔑 配置指南

Trilo 使用小牛翻译 (Niutrans) 的高精度翻译引擎。为了正常使用，你需要配置免费的 API Key。

第一步：获取 API Key

  1. 访问 小牛翻译开放平台 并注册账号。
  2. 登录控制台，点击API应用 开通 文本API服务
  3. 找到你的 APP ID 和 API Key (密匙)。
⚠️ 重要提示：检查免费额度

新注册账号通常会获赠免费流量包。请登录小牛翻译控制台，检查您的剩余积分是否充足。如果额度用尽，API 将无法返回结果。

第二步：在 Trilo 中配置

  1. 在电脑右下角的系统托盘中找到 Trilo 的图标（一个小圆圈）。
  2. 右键点击图标，选择 “设置”。
  3. 在设置面板中，将获取的 APP ID 和 API Key 填入对应输入框。
  4. 点击 “保存” 按钮。

❓ 常见问题

软件是免费的吗？

是的，Trilo 软件本身是完全免费且开源的。小牛翻译 API 也提供了非常慷慨的免费额度，对于个人日常使用通常是足够的。

为什么双击桌面的快捷方式打不开？

Trilo需要先选中文本，然后按下快捷键才会有翻译框弹出

支持哪些操作系统？

Trilo 基于 Tauri 构建，支持 Windows、macOS 和 Linux 全平台。

为什么按快捷键没有反应？

请检查：1. 软件是否在后台运行（托盘区是否有图标）；2. 快捷键是否与其他软件冲突（尝试在设置中更改快捷键）；3. 是否正确配置了 API Key。

如何开机自启动？

在设置面板中，打开“开机自启动”开关即可。
